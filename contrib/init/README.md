Sample configuration files for:
```
SystemD: yarncoind.service
Upstart: yarncoind.conf
OpenRC:  yarncoind.openrc
         yarncoind.openrcconf
CentOS:  yarncoind.init
macOS:    org.yarncoin.yarncoind.plist
```
